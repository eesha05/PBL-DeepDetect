# DeepDetect 🔍
### A Deep Learning Based Image Steganalysis System

DeepDetect is a deep learning-based steganalysis system designed to detect hidden information embedded within digital images. The project focuses on identifying steganographic manipulations using Convolutional Neural Networks (CNNs).

---

## 📌 Project Overview

Steganography is the technique of hiding secret data inside digital media such as images. While it is useful for secure communication, it can also be misused.

DeepDetect aims to:
- Detect whether an image contains hidden data
- Classify clean vs stego images
- Improve detection accuracy using deep learning models
